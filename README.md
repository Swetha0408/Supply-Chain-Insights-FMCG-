# Supply-Chain-Insights-FMCG
Designed an end-to-end Logistics Performance Dashboard using Power BI to monitor order volume, on-time delivery, customer satisfaction, hub performance, driver efficiency, and fleet utilization.

üß† Key Insights
OTIF performance is significantly below target, indicating gaps in fulfillment efficiency.
In-Full % shows room for improvement across multiple customers.
Certain cities demonstrate better On-Time performance but weaker In-Full rates.
Majority of deliveries fall within 0-day delay, but delayed orders still impact service levels.
Specific customers contribute disproportionately to service variance.
Vehicle breakdown frequency increases with vehicle age, indicating maintenance optimization opportunities.
